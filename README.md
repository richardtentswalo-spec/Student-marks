
# Student Marks System (Python)

## Overview
This is a Python program that simulates a simple student marks system. It generates random marks for students, determines their results, and calculates overall statistics.

## Features
- Generates random marks for 15 students
- Determines result categories:
  - Distinction (75+)
  - Pass (50–74)
  - Fail (below 50)
- Calculates:
  - Highest mark and student
  - Lowest mark and student
  - Average mark
  - Total distinctions, passes, and fails
- Displays clear formatted output

## How it works
For each student, the program:
1. Generates a random mark (0–100)
2. Determines the result using conditions
3. Tracks statistics like highest, lowest, and totals

## Concepts Used
- Python functions
- Loops (for loop)
- Conditional statements (if/elif/else)
- Random number generation
- Variables and counters
